# ofertou-site-de-promocoes
Esse é um site de promoções que consome uma API de produtos fictícios chamada DummyJson, o objetivo dessa plataforma é que as pessoas acessem ela, conferem as promoções e se se interessarem, elas compram. A plataforma conta com um sistema de busca, uma home que exibe os produtos por categoria, um sitema de listas de favoritos.
