# ofertou-site-de-promocoes
Esse é um site de promoções que consome uma API de produtos fictícios chamada DummyJson, o objetivo dessa plataforma é que as pessoas acessem ela, conferem as promoções e se se interessarem, elas compram. A plataforma conta com um sistema de busca, uma home que exibe os produtos por categoria, um sitema de listas de favoritos.
##Interface
Para tornar o projeto mais único, criei a uma identidade base e uma interface do zero no Figma, para facilitar o desenvolvimento de todos os componentes e uso de cores, como é um projeto pessoal e ainda estou aprendendo o React, busquei fazer uma interface que fosse bonita, moderna e simples ao mesmo tempo, porque não adiantaria eu criar um layout super complexo e não conseguir implementar no código.
